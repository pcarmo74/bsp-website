Your Stack

Built for your tools,
not ours.

Business Brain ships with example integrations for the most common tools. But every integration is just a TypeScript interface — 80 to 150 lines. You tell Claude Code which tool you actually use, and it writes the adapter for you.

You don't configure anything manually. You describe your stack to Claude Code and watch it wire everything together. Typical adapter takes 1–3 hours of Claude doing the work while you watch.

Example: You run a coaching practice on HubSpot instead of Gmail, take payments via Square instead of Stripe, and send newsletters via ConvertKit instead of Resend. You point Claude Code at the existing adapters and ask it to write yours. About 3–6 hours of Claude working on a Saturday afternoon. Your business now runs on Business Brain. With your tools.

Common Questions

Everything you'd
want to ask.

What if I don't use the integrations that ship with it?
Each integration is a TypeScript interface — 80 to 150 lines. You tell Claude Code which tool you use, and it writes the adapter by copying the existing pattern. You don't write code. You steer Claude Code.
Do I need to know how to code?
You need to be comfortable opening Claude Code, pasting a prompt, and watching it work. If you've ever steered an AI through a codebase, you can do this. You don't write code — Claude does.
What does it cost to run, beyond the purchase?
Claude API usage (Anthropic billing, typically $10–$40/month depending on usage) + a free Supabase account. No other subscriptions. You own the repo — nothing else to pay.
Can I use this for my clients, not just my own business?
Yes. The blueprint can be adapted for client installs. If you want to offer Business Brain as a done-for-you service to your own clients, that's a conversation worth having. Get in touch →
What's the difference between the Blueprint and Done For You?
Blueprint: you run the install yourself with Claude Code doing the heavy lifting. Done For You: Pablo installs everything, connects your tools, and hands you the keys when it's running. Same end state, different path.
How long do I get updates?
12 months of updates included from purchase date. New agents, new adapters, and improvements ship to your repo automatically. After 12 months, you keep everything you have — forever.
Is this different from the BSP practice tools (HelloLobby, Rebook, InboxOS)?
Yes — completely different products for different buyers. HelloLobby and the Practice Bundle are done-for-you SaaS tools for local service businesses. Business Brain is an owner-operated AI OS for operators and builders who want to run Claude Code on their own infrastructure. See the practice tools →
What happens immediately after I buy?
You're added to the GitHub repo within seconds. A welcome email arrives with your start prompt. You open Claude Code, paste the prompt, and the install begins. First agents fire the next morning.
